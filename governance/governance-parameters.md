# Governance Parameters

This section highlights a non-exhaustive list of parameters and actions that the community can govern through the proposed governance mechanism. The pSTAKE community is also encouraged to explore new horizons of governance parameters and actions as pSTAKE expands support for new networks and ecosystems.

#### Protocol Economics[¶](broken-reference) <a href="#protocol-economics" id="protocol-economics"></a>

| Parameters                     | Description                                                                                   | Current Value                         |
| ------------------------------ | --------------------------------------------------------------------------------------------- | ------------------------------------- |
| Minting Fees                   | <p>Fee on minting of stkASSETs as a % of deposit amount<br>(Independent for each network)</p> | 0.00%                                 |
| Service Fees                   | <p>Fee charged as % of staking rewards after commission<br>(Independent for each network)</p> | 0.00%                                 |
| Fee Distribution for verticals | Stakeholders amongst whom fee is distributed                                                  |                                       |
| Fee distribution %             | % distribution of fee amongst verticals                                                       |                                       |
| Reward rate for stkASSETs      | Staking rewards on stkASSETs issued on pSTAKE                                                 | <p>stkATOM: ~12%<br>stkXPRT: ~33%</p> |

#### Snapshot Parameters[¶](broken-reference) <a href="#snapshot-parameters" id="snapshot-parameters"></a>

| Parameters         | Description                                                                                                                                    | Current Value                                                                                                                                                                                                                                       |
